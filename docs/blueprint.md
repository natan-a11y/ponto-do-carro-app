## Core Features:

- Lead Capture: Capture leads via mini-form and WhatsApp integration, storing data in Firestore and triggering notifications.
- Unit Pages: Display unit-specific information (NAP, map, photos, FAQ) with CTAs for WhatsApp and scheduling.
- Appointment Scheduling: Enable users to schedule appointments with a multi-step form, saving data to Firestore and sending notifications.
- Local SEO: Implement structured data (LocalBusiness, FAQPage) and optimized content for each unit to enhance local SEO.
- Review Embedding: Embed Google Reviews to display customer reviews to build trust.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and reliability.
- Background color: Very light blue (#E3F2FD) for a clean and modern feel.
- Accent color: Orange (#FF9800) for call-to-action buttons and important links.
- Headline font: 'Playfair', a modern serif to convey an elegant, fashionable, high-end feel; body font: 'PT Sans' sans-serif for body text, for clear readability and a modern touch.
- Use simple, clear icons to represent the different steps in the car selling process and the different units.
- Minimalist layout with clear CTAs and a sticky CTA bar to encourage conversions.
- Subtle animations and transitions to improve user engagement and provide feedback on interactions.
